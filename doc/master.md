# Master

## RoleAbstract

Disini ada hal penting yaitu kita bisa membedakan property pada class nya, untuk membedakannya tinggal merubah pada function syncMemory() pada class Master. Atau lebih tepatnya pada master_definitions.cpp di daerah atas-atas. Property yang di-assign pointernya disana akan menjadi shared property.
**Shared property** disini artinya adalah ketika saya ganti role, maka variable ini tidak berubah. Kalau **Local property** ini akan berubah sesuai pada role-nya.
